# クライアント様向けドキュメント

このディレクトリは、**クライアント様専用**のドキュメントサイトです。

## アクセスURL

**https://yasushi-honda.github.io/doc-split/client/**

## 特徴

- ✅ **完全分離**: 開発者向けドキュメント（ADR、CLAUDE.md等）は非表示
- ✅ **独立サイト**: 独自のサイドバー・検索機能
- ✅ **クライアント専用コンテンツ**: GCPプロジェクトセットアップガイドのみ

## ファイル構成

```
docs/client/
├── index.html          # 独立したDocsifyサイト
├── _sidebar.md         # クライアント専用サイドバー
├── client-setup.md     # GCPプロジェクトセットアップガイド
└── README.md           # このファイル
```

## リンク関係

- **開発者向け → クライアント向け**: リンク有り（開発者はクライアント向けページを参照可能）
- **クライアント向け → 開発者向け**: リンク無し（クライアントは開発者向けページにアクセス不可）

## メンテナンス

クライアント向けコンテンツを追加する場合：
1. `docs/client/` 配下にMarkdownファイルを作成
2. `docs/client/_sidebar.md` にリンクを追加
3. `docs/client/client-setup.md` から必要に応じてリンク

**注意**: 開発者向けドキュメントへのリンクは絶対に追加しないこと。
